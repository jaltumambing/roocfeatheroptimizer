How Filter Builder works Set minimum values for the stats you need. Only builds that meet all your filters are shown.

Leave a field blank to ignore that stat entirely.

Results are ranked by feather efficiency — how much stat you get per copy spent. Secondary rank: total surplus beyond your minimums.

If no builds pass all filters, you'll see which filters had zero qualifying builds.
⚠ Only Raven's Hour set data loaded. Builds failing any filter are excluded entirely — no partial matches.
